# GOJO BOT ⚡

A powerful **Jujutsu Kaisen themed WhatsApp bot** inspired by the strongest sorcerer **Satoru Gojo**.

GojoBot combines **AI intelligence, educational tools, group management systems, media utilities, and entertainment features** to create a powerful WhatsApp automation assistant.
